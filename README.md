#**student-study-tracker**

A simple command-line task management application built using Python.  
This project allows students to manage their study tasks efficiently with automatic saving and confirmation features.

##**Features**
- Add new tasks
- View all tasks with completion status
- Mark tasks as completed
- Delete tasks with confirmation
- Automatic saving using JSON (persistent storage)

##**Technologies Used**
- Python
- JSON for data persistence

##**How It Works**
The program runs in the terminal and presents a menu-based interface.  
Tasks are stored in a JSON file so they are preserved even after the program is closed.

##**How to Run**
1. Make sure Python is installed
2. Clone the repository or download the files
3. Run the program:
   ```bash
   python task_tracker.py
